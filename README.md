A simple little solver for [Spelling Bee](https://www.nytimes.com/puzzles/spelling-bee)
 that uses mit's wordlists:
https://www.mit.edu/~ecprice/wordlist.10000
https://www.mit.edu/~ecprice/wordlist.100000

And finds words that have characters.

# Example
cargo run --release -- g e n l a c d 

g is the must have in each word and the others are letters allowed
